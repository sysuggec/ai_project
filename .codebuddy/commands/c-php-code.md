# PHP 代码编写命令

## 命令说明

当用户请求编写 PHP 代码时，自动调用 `php-coder` 技能执行标准化工作流程。

## 执行方式

使用 `use_skill` 工具调用技能：

```
技能名称: s-php-coder
技能位置: .codebuddy/skills/s-php-coder
```

技能将自动执行完整的工作流程，包括：
- 规则文件引用
- 设计、编码、注释、审查流程
- 代码模板应用
- 质量检查清单

## 使用示例

```
用户: "帮我写一个用户服务类"
用户: "创建一个订单控制器"
用户: "实现一个支付处理功能"
```

---

**详细的工作流程、代码模板和检查清单都在 `s-php-coder` 技能中定义。**
